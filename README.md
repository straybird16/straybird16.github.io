# straybird16.github.io

Personal academic/research website for Xiangdong (Peter) Xie.

## Structure

- `index.md`: homepage
- `projects.md`: selected projects
- `_config.yml`: Jekyll/GitHub Pages configuration
- `assets/css/style.scss`: style overrides
- `assets/files/`: CV and downloadable files
- `assets/img/`: images

## Notes

This site is intentionally minimal:
- concise homepage
- selected work only
- no blog unless it is updated regularly
- no duplicated resume content
